# newbweb-es6-recap
A simple Babel+ESLint repo for testing the ECMAScript 6 code examples found in various lectures.


## Usage
This repo has been created for newBweb students so they can follow along with the ECMAScript 6 recap lectures

Simply replace the code

```JavaScript
helloWorld(); // Replace this with your  ECMAScript 6 test code here
```
with the code from the lectures, and you'll be able to follow along!
